# MarkdownDeck

MarkdownDeck turns one Markdown file into a native-macOS presentation with beautiful PDF export. A deck is plain Markdown — `---` on its own line splits slides, and an optional header at the top configures the whole deck.

## Install

```bash
brew install --cask slantedt/tap/mddeck
```

Requires macOS 14 (Sonoma) or later. Then head to [Getting started](01-getting-started.md) to open your first deck.
